平均强度: 平均强度会在65% - 85%这个范围内变化
总次数: 总次数随着平均强度的增加而减少

负荷计算方法: (A动作强度% * 次数 + B动作强度% * 次数) / 总次数
注意: 用这种方式计算强度时不包含次要动作, 只包含 深蹲, 卧推等主要动作
